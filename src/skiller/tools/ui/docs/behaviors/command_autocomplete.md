# Command Autocomplete

### Trigger

The user types a slash-command prefix in the command input.

### Scope

Autocomplete applies only to command names that start with `/`.

### Command Input

The command input remains editable while autocomplete suggestions are available.

### Autocomplete Menu

The autocomplete menu appears near the input area and shows matching command names.

### Navigation

The user can move through available suggestions with the keyboard.

### Selection

Selecting a suggestion inserts the full command name into the command input.

### Validation

Autocomplete must not suggest plain-text replies or command arguments.

### Result

The command input is updated with the selected command name and remains ready for further editing or submission.
